## Overview

`URecordComponent`는 플레이어의 위치 데이터 저장 및 복구 기능을 제공합니다. 예) "플레이어 위치를 자동으로 저장하고, 게임 시작 시 복원."

Unreal Engine 플러그인에서 핵심 기능을 담당하는 컴포넌트 클래스입니다. 게임 오브젝트에 추가해 XYZ 기능을 사용할 수 있습니다.
